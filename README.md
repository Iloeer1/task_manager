The task manager app allows users to view, add, and delete tasks.
Each task consists of a title, description, and user that the task belongs to.
The setup forms two test users: alice (password: redqueen) and bob (password: squarepants)

1.	(Install Django)
3.	Run migrations "python manage.py migrate"
4.	Start the server "python manage.py runserver"
5.	Access the application at http://127.0.0.1:8000/tasks/
6.	The task manager has two test subjects: alice (password: redqueen) and bob (password: squarepants). Alice starts with two tasks and bob with one.
